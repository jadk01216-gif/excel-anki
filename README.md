# 劍橋字典Excel表格轉apkg檔案 v0.0.3

這是一個將 Excel 單字表轉換為 Anki (.apkg) 格式的 PyQt6 應用程式。

## 功能特色

- **自動翻譯**：使用 `deep-translator` 自動補補全缺失的中文翻譯（繁體）。
- **資料自動填寫**：自動從 Free Dictionary API 獲取詞性與英文解釋。
- **自定義顯示**：可自由選擇是否顯示翻譯、詞性或英文解釋。
- **TTS 支援**：支援 Anki 原生英語發音。
- **填空模式**：自動生成填空式字卡。
- **自動記憶**：程式會記住最後一次匯出的資料夾路徑。
- **現代化設計**：基於 `PyQt6-Fluent-Widgets` 的 Windows 11 風格介面。

## 安裝需求

```bash
pip install -r requirements.txt
```

## 使用說明

1. 準備一個 Excel 檔案（`.xlsx` 或 `.xls`）。
2. 第一行為標題橫幅，第二行為欄位名稱。
3. 第一欄為單字，後續欄位程式會自動處理或補充。
4. 執行 `python main.py` 開啟程式。
5. 選擇 Excel 檔案，設定牌組名稱與選項，點擊「轉換為 APKG」。
